# ICON Global (ICOsahedral Nonhydrostatic)

## What this model is
A global weather forecast model that predicts weather conditions such as temperature, wind, precipitation, and pressure for the entire Earth.

---

## Who runs it
- **Organization:** : DWD (Deutscher Wetterdienst)
- **Country / region:** Germany

---

## What area it covers
- **Coverage:** Global

---

## Basic details
- **Model type:** Global
- **Typical resolution:** ~13 km
- **Forecast length:** Up to 7 days
- **Update frequency:** 4× daily

---

## Data availability
- **Is the data free?** Yes
- **Is the data downloadable?** Yes
- **Data formats:** GRIB2
- **Official download location:**  
  https://opendata.dwd.de/weather/nwp/icon/

---

## Notes
ICON Global is Germany’s primary global weather forecast model and uses a modern non-hydrostatic core with an icosahedral grid.
